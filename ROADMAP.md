# Roadmap & Ideas

## Visual improvements for Instagram posts

Recommendations based on visual review of generated slides (April 2026).

### Content slides — high impact

- [ ] **Pull quotes / accent-colored key phrases**: Highlight a key sentence per slide in the brand gold/beige accent color to break up text walls and give the eye an anchor point.
- [ ] **Slide numbering**: Add a progress indicator (e.g. "2/5") so users know how many slides remain — encourages swiping through.
- [ ] **Stronger continuation indicator**: The current ">" is too discreet. Use a more visible swipe cue (arrow icon, or the page number itself).
- [ ] **Opening line emphasis**: Use a larger or bolder style for the first line of each content slide to create a visual hook.
- [ ] **Vary content slide backgrounds**: Slightly shift the background tint per slide (warmer, cooler) or add a subtle accent stripe to prevent the "every slide looks the same" feeling.

### Title slide (slide-0)

- [ ] **Stronger gradient overlay**: On busy photos the title text competes with the background. A more intentional gradient (or semi-transparent color band behind the title) would improve legibility.
- [ ] **Show intro/subtitle**: Currently no teaser text appears below the title — a short intro line in a lighter weight would add visual layering and fill the empty space.

### Final CTA slide

- [ ] **Bolder CTA**: Make "Abonne-toi !" larger, possibly with a button-like shape or stronger use of the gold accent color — this is the conversion moment.
- [ ] **Visual callback to cover image**: Reuse the article's photo (heavily darkened/blurred) on the last slide to create a visual bookend.
- [ ] **Differentiate per article**: Currently every post's final slide looks identical regardless of the article's mood.
